Демонстрация уязвимостей

Этот сайт содержит намеренно созданные уязвимости для обучения:

1. SQL-инъекции

На страницах Поиск и Вход реализованы SQL-инъекции.

Пользовательский ввод напрямую подставляется в SQL-запросы без проверки.

2. XSS (Межсайтовый скриптинг)

На странице Комментарии реализована XSS уязвимость.

Пользовательский ввод отображается без экранирования HTML.

Безопасные версии

Также доступны безопасные версии этих страниц:

  • Безопасный поиск (параметризованные запросы)
  • Безопасные комментарии (экранирование HTML)

Исходный код

Полный исходный код этого демонстрационного сайта доступен в файле main.py.
