Демонстрация SQL-инъекций и XSS уязвимостей
Этот репозиторий содержит примеры уязвимого и безопасного кода, а также демонстрационный веб-сайт для изучения SQL-инъекций и XSS (межсайтового скриптинга).

Запуск демонстрационного сайта
Убедитесь, что у вас установлен Python 3.7 или выше и FastAPI

Перейдите в папку website:

bash
`cd website`
Установите зависимости:

bash
`python main.py`
Откройте браузер и перейдите по адресу: http://localhost:8000

Содержание папок
SQL-Injection
Содержит примеры кода, демонстрирующие:

Уязвимые SQL-запросы, подверженные инъекциям

Безопасные методы (параметризованные запросы, подготовленные выражения)

Примеры атак и способы защиты

XSS
Содержит примеры кода, демонстрирующие:

Reflected XSS, Stored XSS и DOM-based XSS

Небезопасную обработку пользовательского ввода

Методы защиты (экранирование, Content Security Policy)

Website
Интерактивный демонстрационный сайт, который показывает:

Работающие примеры SQL-инъекций

Различные типы XSS атак

Сравнение уязвимого и безопасного кода

Практические примеры эксплуатации и защиты
