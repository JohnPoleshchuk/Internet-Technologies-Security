Демонстрация SQL-инъекций и XSS уязвимостей
Этот репозиторий содержит примеры уязвимого и безопасного кода, а также демонстрационный веб-сайт для изучения SQL-инъекций и XSS (межсайтового скриптинга).

Структура репозитория
text
├── SQL-Injection/          # Примеры кода с SQL-инъекциями
│   ├── vulnerable/         # Небезопасный код
│   └── secure/            # Безопасный код
├── XSS/                   # Примеры кода с XSS уязвимостями
│   ├── vulnerable/        # Небезопасный код
│   └── secure/           # Безопасный код
└── website/              # Демонстрационный веб-сайт
    ├── main.py           # Основной файл для запуска сервера
    ├── requirements.txt  # Зависимости Python
    ├── templates/        # HTML шаблоны
    └── static/          # Статические файлы
Запуск демонстрационного сайта
Убедитесь, что у вас установлен Python 3.7 или выше

Перейдите в папку website:

bash
cd website
Установите зависимости:

bash
pip install -r requirements.txt
Запустите сервер:

bash
python main.py
Откройте браузер и перейдите по адресу: http://localhost:5000
